#  3D Printed Gear Pendant / Keychain

A clean and simple 3D-printable spur gear pendant designed using **Onshape**. This model is optimized for 3D printing and serves as a stylish, lightweight keychain or jewelry pendant.

---

##  Design Specifications
The gear has been meticulously modeled to ensure a balance between aesthetics and durability:
* **Outer Diameter:** `43.00 mm`
* **Root Diameter:** `35.00 mm`
* **Bore (Hanging Hole) Diameter:** `2.00 mm` *(optimized for fine chains or small keyrings)*

---

##  Recommended 3D Printing Settings
For the best physical result, we recommend the following slicer settings:
* **Suggested Materials:** PLA, PETG (for lightweight plastic) or Aluminum (for CNC machining/metal print).
* **Layer Height:** `0.12 mm` to `0.20 mm` (for smooth and clean gear teeth).
* **Infill:** `20%` or higher.
* **Supports:** No supports needed (the design is perfectly flat and easy to print).
